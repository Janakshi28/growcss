Tabs

Tabs are elements that help you organize and navigate multiple documents in a single container. They can be used for switching between items in the container.

No active tab

active="0" Means that no tab is active.

active="2" sets the second tab as initially active, etc.

Fullwidth Tabs

Tabs are not wrapped in grid-container and grid-flex.

Mode

mode="accordion" The tabs themselves occupy the full width.

Mode

mode="vertical" The tabs are placed on the left side.

Mode

mode="thumbnail" Responsive thumbnail tabs. Automagically converted to an accordion in the small layout.
