Base

Provides the default style for all HTML elements.

This component defines the base look of your page. It offers great typography by setting the default colors, margins, font-sizes and more for each HTML element. This page is a short guide on how to use basic HTML elements and how GrowCss styles them.
Basically GrowCss utilizes the famous Normalize.css. This makes GrowCss and its components very robust against conflicts with other CSS frameworks so that you don't have to worry about browser inconsistencies in your projects.
